# collezione-film
progetto tpsit

Installazione ed esecuzione

Avvia XAMPP (o ambiente equivalente) con Apache e MySQL attivi
Copia la cartella collezione_film/ in htdocs/
Apri phpMyAdmin e importa lo script SQL qui sopra
Verifica le credenziali in db.php (host, porta, utente, password)
Apri il browser e vai su http://localhost/collezione_film/


Nota: il file db.php è configurato con port=3307. Se il tuo MySQL usa la porta standard, cambia in port=3306.
